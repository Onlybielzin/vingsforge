# Sidecar binaries

The compiled `@vingsforge/sidecar` is placed here as a Tauri external binary,
named `vingsforge-sidecar-<target-triple>` (e.g. `vingsforge-sidecar-x86_64-unknown-linux-gnu`).

It is produced by packaging the Node sidecar into a single executable
(Spec 09 §4: `bun build --compile` / Node SEA / `pkg`). Not generated yet.
